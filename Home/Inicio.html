DefiniciónImportante

Es un sistema operativo open source. El sistema operativo es el software que gestiona directamente el hardware de un sistema y sus recursos, como la CPU, la memoria y el almacenamiento. Se encuentra entre las aplicaciones y el hardware, y establece las conexiones entre todos los sistemas de software y los recursos físicos que ejecutan las tareas.
