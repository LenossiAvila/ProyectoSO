L I N U X



Características

Código abierto

El código fuente del kernel Linux y la mayoría de sus componentes son accesibles y modificables por cualquier persona bajo licencias de software libre, lo que fomenta la colaboración y la mejora continua.

Multiplataforma

Linux es compatible con una amplia variedad de arquitecturas de hardware, desde computadoras de escritorio y servidores hasta dispositivos integrados y supercomputadoras.

Estabilidad y fiabilidad

Linux es conocido por su estabilidad y fiabilidad. Los sistemas Linux son utilizados ampliamente en servidores y sistemas embebidos donde la estabilidad es crucial.

Seguridad

Linux ofrece un sólido sistema de seguridad con características como permisos de usuario, control de acceso obligatorio (MAC), cortafuegos, y actualizaciones regulares para abordar vulnerabilidades de seguridad.

Flexibilidad y personalización

Linux es altamente configurable y adaptable a las necesidades del usuario. Los usuarios pueden elegir entre una amplia variedad de distribuciones (distros) Linux, cada una con sus propias características y conjuntos de software preinstalado. Más información

Gran comunidad de desarrolladores

Linux cuenta con una gran comunidad de desarrolladores y usuarios que contribuyen con mejoras, correcciones de errores y soporte técnico en foros y listas de correo.

Software libre y de código abierto

La mayoría del software disponible para Linux es de código abierto y gratuito, lo que permite a los usuarios acceder, estudiar, modificar y distribuir el software según sus necesidades.

Rendimiento

Linux es conocido por su eficiencia y buen rendimiento, especialmente en entornos de servidor y en sistemas embebidos.

Abstracción del proceso

En Linux, un proceso es una abstracción fundamental que representa la ejecución de un programa. Esto implica que múltiples programas pueden ejecutarse simultáneamente como procesos independientes.



Una distribución de un sistema operativo es una versión particular del sistema operativo que se ha adaptado y configurado para satisfacer las necesidades de un grupo específico de usuarios o para un propósito particular. Existen muchas distribuciones (distros) de Linux, cada una adaptada para diferentes propósitos y preferencias de los usuarios. Algunas de las distribuciones más populares y conocidas basadas en el kernel Linux son las siguientes.



Ubuntu

Una de las distribuciones más populares, conocida por su facilidad de uso y su amplia base de usuarios. Viene en diferentes sabores, como Ubuntu Desktop, Ubuntu Server, Ubuntu Mate, entre otros.

Debian

Una distribución conocida por su estabilidad y su compromiso con el software libre. Debian es la base de muchas otras distribuciones populares, incluyendo Ubuntu.

Fedora

Desarrollada por la comunidad con el respaldo de Red Hat, Fedora es conocida por su enfoque en la innovación y la adopción temprana de nuevas tecnologías.

Red Hat Enterprise Linux (RHEL)

Una de las mejores distribuciones de Linux disponibles, RHEL es una distribución potente, estable y segura desarrollada por Red Hat basada en Fedora. Está disponible en la nube y en plataformas IoT y puede gestionar grandes datos, visualización y contenedores de datos.

openSUSE

Una distribución mantenida por una comunidad y respaldada por SUSE Linux GmbH. Ofrece diferentes ediciones, incluyendo una orientada al escritorio y otra para servidores.

CentOS

Basada en el código fuente de Red Hat Enterprise Linux (RHEL), CentOS es una distribución popular entre los servidores debido a su estabilidad y compatibilidad con RHEL.

Arch Linux

Una distribución ligera y altamente personalizable, dirigida a usuarios avanzados que prefieren configurar su sistema desde cero.

Linux Mint

Basada en Ubuntu, Linux Mint se centra en proporcionar una experiencia de usuario familiar y amigable, con una selección de software adicional preinstalado.

Gentoo

Conocida por su enfoque en la optimización y personalización, Gentoo permite a los usuarios compilar su sistema desde el código fuente para adaptarse a sus necesidades específicas.
